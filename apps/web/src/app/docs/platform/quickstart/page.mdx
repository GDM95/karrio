# Quickstart

The Karrio Platform is our managed cloud service that provides all the benefits of Karrio without the overhead of managing your own infrastructure.

## Features

### Fully Managed Infrastructure

- **High Availability**: 99.9% uptime SLA for business and enterprise plans
- **Automatic Scaling**: Resources automatically scale based on your needs
- **Global CDN**: Fast access from anywhere in the world
- **Regular Backups**: Automated backups to prevent data loss
- **Security**: SOC 2 compliant infrastructure with encryption at rest and in transit

### Carrier Integrations

- **Pre-configured Carriers**: All supported carriers are pre-configured and ready to use
- **Automatic Updates**: Carrier API changes are handled automatically
- **New Carriers**: Regular additions of new carrier integrations

### Developer Tools

- **API Keys**: Create and manage API keys for different environments
- **Webhooks**: Configure webhooks for real-time event notifications
- **Logs**: Access detailed logs for troubleshooting
- **Testing Tools**: Sandbox environment for testing integrations

## Plans

### Starter

Perfect for small businesses and startups.

- Up to 500 shipments per month
- 2 user accounts
- Email support
- Basic analytics
- Standard API rate limits

### Business

Ideal for growing businesses with moderate shipping volume.

- Up to 10,000 shipments per month
- 10 user accounts
- Priority email support
- Advanced analytics
- Higher API rate limits
- Custom webhooks

### Enterprise

For high-volume shippers and large organizations.

- Unlimited shipments
- Unlimited user accounts
- Dedicated support manager
- Phone and email support
- Custom SLAs
- Highest API rate limits
- Custom integrations
- On-boarding assistance

## Getting Started

### 1. Create an Account

Sign up at [app.karrio.io](https://app.karrio.io/signup) to create your Karrio Platform account.

### 2. Configure Your Account

- Add your business information
- Set up your first user accounts
- Configure your default settings

### 3. Connect Carriers

- Add your carrier credentials
- Test connections with sample shipments
- Set up carrier-specific settings

### 4. Integrate with Your Systems

- Generate API keys
- Review the [API documentation](/docs/api-v2-reference)
- Set up webhooks for real-time updates

## FAQ

### Can I migrate from self-hosted to Karrio Platform?

Yes, we provide migration tools and assistance to help you move from a self-hosted Karrio instance to the Karrio Platform.

### Is my data secure?

Yes, all data is encrypted at rest and in transit. We follow industry best practices for security and are SOC 2 compliant.

### Can I export my data?

Yes, you can export your data at any time through our API or dashboard.

### Do you offer custom integrations?

Custom integrations are available for Enterprise customers. Contact our sales team for more information.

### How do I upgrade my plan?

You can upgrade your plan at any time through the billing section of your dashboard.
